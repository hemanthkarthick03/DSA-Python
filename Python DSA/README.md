## üîë Key Patterns to Master

### 1. **Two Pointers**
```
Problems: Two Sum, Three Sum, Container With Water
Time: O(n) | Space: O(1)
```

### 2. **Sliding Window**
```
Problems: Longest Substring, Max Subarray, Min Window
Time: O(n) | Space: O(k)
```

### 3. **Fast & Slow Pointers**
```
Problems: Cycle Detection, Middle of List, Palindrome List
Time: O(n) | Space: O(1)
```

### 4. **BFS/DFS**
```
Problems: Trees, Graphs, Islands, Course Schedule
Time: O(V + E) | Space: O(V)
```

### 5. **Dynamic Programming**
```
Problems: Fibonacci-like, Knapsack, LCS, Edit Distance
Time: O(n¬≤) | Space: O(n) optimized
```

### 6. **Backtracking**
```
Problems: Subsets, Permutations, N-Queens
Time: O(2^n) or O(n!) | Space: O(n)
```

### 7. **Binary Search**
```
Problems: Search in Rotated, Find Min, Sqrt
Time: O(log n) | Space: O(1)
```
